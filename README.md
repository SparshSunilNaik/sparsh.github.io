# SparshOS Portfolio

Interactive retro terminal / fake operating system portfolio built with Next.js, React, and Tailwind.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Edit content

Most portfolio text lives in:

- `lib/portfolioData.ts` — profile, contacts, projects, filesystem lore
- `public/resume.pdf` — add your real resume here
- `public/placeholders/` — replace placeholder visuals with real project images

## Terminal commands

Try: `help`, `about`, `skills`, `projects`, `project egonav`, `ls`, `cd /projects`, `cat README.md`, `neofetch`, `secret`, `hack`.

Keyboard support:

- `Tab` autocompletes commands, project names, files, and folders.
- `↑` / `↓` move through command history.
